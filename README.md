# KayJustus

Minimalistische homepage voor het personal brand van Kay Justus.

## Lokaal runnen

Omdat dit een statische pagina is kun je direct openen in je browser:

1. Open `index.html`.

Of start een simpele lokale server vanuit deze map:

```bash
python3 -m http.server 8000
```

Ga daarna naar `http://localhost:8000`.
